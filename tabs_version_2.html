Home

This is the default tab shown when the page loads.

First Tab

  • This panel uses the shared `.tab-panel` class.
  • Only the active panel is visible.

Second Tab

  • Each button stores its target panel inside `data-tab`.
  • This keeps the HTML cleaner than hardcoding many functions.

Third Tab

  • The active button gets a different style.
  • This helps the user know which tab is selected.

Fourth Tab

  • The JavaScript loops through buttons and panels.
  • Then it adds or removes the `active` class when needed.
